# Customer Churn Analysis
Project untuk menganalisis pola pelanggan yang berhenti menggunakan layanan (churn) dan memberikan rekomendasi bisnis untuk meningkatkan retensi.

Tools:
Python (Pandas, Matplotlib, Seaborn)

Tujuan:
Menemukan faktor yang paling mempengaruhi churn dan menyusun strategi retensi pelanggan.

Insight utama:
- Kontrak bulanan memiliki churn tertinggi
- Pelanggan churn cenderung memiliki biaya bulanan tinggi
- Pelanggan baru dengan biaya tinggi sangat berisiko churn

Rekomendasi bisnis:
- Upgrade pelanggan kontrak bulanan ke kontrak tahunan
- Program onboarding pelanggan baru 3 bulan pertama
- Penawaran bundling layanan untuk meningkatkan perceived value

- Notebook.ipynb
- Report.pdf
